Here are some of the projects I've built lately, whether for money, learning, or just for fun.

RegExperiment

A clone of RegExceptional, a HackReactor final project that consists of a matching game to help teach regular expressions. My goal is to show that we can get exactly the same functionality without any frameworks and that the resulting site will not only look nearly identical, but will also load less data and do so faster. The clone I built reduces the initial data transfer by 99.83%, and cuts the page load time from 600ms to right around 100ms, which seems like a success.
